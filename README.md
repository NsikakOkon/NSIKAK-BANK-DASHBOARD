# NSIKAK-BANK-DASHBOARD
This repository contains a professional Power BI dashboard designed to analyze and visualize key metrics from a banking dataset. The dashboard provides actionable insights into customer demographics, deposits, loans, transactions, and branch performance, enabling data-driven decision-making for financial institutions.
Features
• 	Customer Demographics
• 	Age, gender, occupation, and income distribution
• 	Geographic segmentation across branches and regions
• 	Deposits Analysis
• 	Total deposits by product type (savings, current, fixed deposit)
• 	Monthly and yearly growth trends
• 	Average account balance per customer segment
• 	Loan Portfolio
• 	Loan disbursement by category (personal, mortgage, business)
• 	Repayment vs. default rates (NPL ratio)
• 	Interest income contribution
• 	Transaction Insights
• 	Transaction volume and value across channels (ATM, online, branch)
• 	Peak transaction times and seasonal trends
• 	Fraud detection indicators
• 	Branch Performance
• 	Revenue and profitability by branch
• 	Customer acquisition and retention rates
• 	Regional comparisons
• 	Customer Retention & Churn
• 	Active vs. inactive accounts
• 	Attrition analysis by segment
• 	Cross-selling opportunities
Dashboard Design
• 	Professional banking theme (navy blue, black, white, green)
• 	KPI cards for high-level metrics (Total Deposits, Loan Portfolio, NPL Ratio, Active Customers)
• 	Interactive filters for branch, product type, and time period
• 	Drill-down capabilities with tooltips for detailed analysis
• 	Clean and minimal layout for clarity and usability
Purpose
The goal of this dashboard is to empower banking professionals with clear, data-driven insights that support strategic planning, risk management, and customer engagement initiatives.
